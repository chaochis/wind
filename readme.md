# 风旅博客项目

**模块说明:**
1. wind-b-service 用于储存数据调取数据的服务
2. wind-b-web 提供所有的api接口
3. wind-f-admin 风旅博客的管理后台的前端项目
4. wind-f-web 风旅博客的pc端
5. wind-f-mobile 风旅博客的移动端
6. wind-f-uni 风旅博客的小程序端

**运行:**

1. 后台api接口是wind-b-web, 用idea运行时需要在jvm
